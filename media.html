In this Page, I will be making use of all the media elements in the HTML and making sure that they're responsive to different screen models.

1. Audio Element: <audio src=""></audio> It allows us to add audio content such as music or sound files.

2. Video: <video src=""></video> This is used to add video content to the webpage. It can include multiple source elements to support different video.

3. Image Element: <img> The Image elemet is used to add images to the webpage and it is a self closing tag.

4. Picture Element: <picture></picture> The picture elemet is used to define multiple sources of an image. This allows the browser to choose the most appropriate image based on the device or screen especially when you're trying to achieve a responsive web design. You can test this out by shrinking your web browser.
